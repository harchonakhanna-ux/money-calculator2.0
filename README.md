# Money Redistribution Calculator

An interactive calculator that shows how billionaire luxury spending could be redirected to help people in need.

## Features

- Calculate how many people could be fed with money spent on luxury items
- Calculate how many people could be vaccinated ($31 per vaccine)
- See how many schools, hospitals, and wells could be built
- Visual representation of the impact
- Preset examples of billionaire spending

## How to Use

1. Enter an amount in US dollars
2. Or click one of the preset buttons for common luxury items
3. See the results instantly update

## Calculations

- Food: $3.54 per person per day
- Vaccinations: $31 per person
- Schools: $300,000 per school
- Hospitals: $2,000,000 per hospital
- Wells: $5,000 per well
